# Blog-Web-App


## Description

A Daily journal app for keeping track of your thoughts.

## Installation

Download the files and install npm.

```bash
npm i
```

Also install Nodemon to have an active server initialized.
```bash
npm install -g nodemon
```
Run the app
```bash
nodemon app
```

type http://localhost:3000/ in your browser adress bar


## Usage

Add entries to your daily journal.
